Introduction

Python is a popular high-level, versatile programming language known for its simplicity and readability. It has a wide range of uses from web development, to data science and artificial intelligence, etc.

Some key characterstics and features include:

  1. Readibility: Python's syntax is easily readable, and resembles the English language structure. The readibility reduces the cost of development and maintenance.
  2. Versatility: Python supports multiple programming paradigms, such as object-oriented, and functional, giving developers a variety of approaches to reach a solution for their projects.
  3. Interpreted & Interactive: As an interpreted language, code is executed line-by-line by an interpreter, enabling quick development and debugging. The language comes with interactive shells for testing code.
  4. Rich Standard Library: Python contains a vast library of modules and functions as standard, reducing the need for external modules.
  5. Extensive Ecosystem: Third party packages and frameworks can be leveraged by developers. These include Django and Flask for web developments, NumPy and Pandas for data analysis, Tensorflow and PyTorch for machine learning, and more.
  6. Platform Independence: Largely platform independent code means Python programs is compatible on various operating systems.
  7. Community & Support: The Python community is large and active, contributing to its growth, with support coming from numerous resources and documentation.
  8. Open Source: Python is open source, encouraging developers to collaborate, contribute, expand, and improve.

Variables

You can store values by assigning them to a variable. Variables are useful for values that are required on multiple occassions.

Variables should follow a naming convention commonly used in Python. This may be in camelCase, where the initial letter is lowercase, and the following words are capitalised. However, according to PEP 8 Style Guide for Python, variables (and functions) should follow the snake_case, unless camelCase is required for context to retain backwards compatibility.

An appropriate identifier should be concise and clear as to the purpose of the variable for future reference.

An example is given below:

house_number = 11
Data Types

Python can store various data types in variables:

  • Text: str
  • Numeric: int, float, complex
  • Sequence: list, tuple, range
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview
  • None: NoneType
